## 학습순서

1. man page 읽고 이해. ([FreeBSD manual](https://www.freebsd.org/cgi/man.cgi)을 참고함)
2. google, 이해가 어려우면 한글자료도 검색.
3. 그럼에도 막히는 부분이 있다면,
   1. Slack, Discord, Stackoverflow 등등 검색.
   2. Kakaotalk, 동료에게 질문.
4. 간단히 코드 작성 및 수정.
5. test case를 구상, Undefined Behavior를 어떻게 처리할 것인지 고민.
6. 3가지 테스트기 실행.
   - 실패시 다시 4번으로.
7. 느낀 점, 배운 것 기록.
